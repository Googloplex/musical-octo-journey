# musical-octo-journey
Плагин для [Obsidian](https://obsidian.md), реализующий метод конспектирования **Cornell Notes** — трёхпанельный редактор с разделами «Вопросы / Идеи», «Конспект» и «Резюме».  Каждая секция хранится как отдельный `.md` файл — полная поддержка wikilinks, embeds, LaTeX, backlinks, graph view и поиска.
